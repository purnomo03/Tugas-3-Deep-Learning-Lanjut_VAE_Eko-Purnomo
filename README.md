Tugas: Individu (Variational Autoencoder).

Tugas ini merupakan implementasi Variational Autoencoder (VAE) menggunakan dataset Fashion-MNIST. Berbeda dengan Autoencoder biasa yang memetakan input ke satu titik tetap, VAE memetakan input ke dalam distribusi normal multivariat di latent space , memungkinkan generasi data baru yang lebih halus dan kontinu.
